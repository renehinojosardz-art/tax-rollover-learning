# Tax Rollover Learning

Static HTML learning workspace for federal tax structure fluency in M&A rollovers, equity contributions, LLCs, partnerships, C-corps, recapitalizations, and reorganizations.

Open `index.html` as the site entry point.

Personal answer logs should live in `private/`, which is intentionally ignored by git because this repository is published publicly through GitHub Pages.

The live HTML lessons also include a client-side answer logger. It stores answers in the browser's local storage and can copy/export Markdown packets for review. It does not write to GitHub or transmit answers automatically.
